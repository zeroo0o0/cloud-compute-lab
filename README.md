# CloudComputeBookCode

云计算课程配套代码仓库，包含课程示例代码和实验代码，用于课堂演示、实验练习和课程实践。

## 仓库结构

```text
.
├── CourseCode
│   ├── ch3
│   ├── ch4
│   ├── ch5
│   └── ch6
├── Lab
│   ├── Lab1
│   ├── Lab2
│   ├── Lab3
│   └── Lab4
├── .gitignore
└── README.md
```

## 目录说明

* `CourseCode/`：课程章节配套代码，按章节目录组织。
* `Lab/`：课程实验代码，按实验编号组织。
* `.gitignore`：Git 忽略规则。
* `README.md`：仓库说明文档。

## 使用方式

克隆仓库：

```bash
git clone https://gitee.com/hnu-cloudcomputing/cloud-compute-book-code.git
cd cloud-compute-book-code
```

进入对应章节或实验目录，根据目录内代码和说明运行。

例如：

```bash
cd CourseCode/ch3
```

或：

```bash
cd Lab/Lab1
```

## 说明

本仓库主要用于云计算课程教学与实验实践。不同章节和实验目录可能包含独立的运行方式，请以对应目录下的代码和说明为准。
